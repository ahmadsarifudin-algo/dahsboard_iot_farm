## Run Frontend Dev Server

// turbo-all

1. Install dependencies (if needed):
```bash
cd c:\Users\Chickin\.gemini\antigravity\playground\dashboard-iot\frontend
npm install
```

2. Start the development server:
```bash
cd c:\Users\Chickin\.gemini\antigravity\playground\dashboard-iot\frontend
npm run dev
```

3. Open http://localhost:3000 in the browser.
